🎓 SGPA & CGPA Calculator

A simple and interactive web-based calculator to compute SGPA (Semester Grade Point Average) and CGPA (Cumulative Grade Point Average) based on the grading system.

---

🚀 Features

- 📊 Two Input Modes
  
  - Free Style → Enter obtained marks + total marks
  - Detailed Style → Enter ISE 1, ISE 2, MSE, ESE

- 🧮 Automatic SGPA Calculation
  
  - Uses weighted formula:
    "SGPA = Σ(C × G) / Σ(C)"

- 🎯 CGPA Calculation
  
  - Based on multiple semesters
  - Uses weighted formula (not simple average)

- ⚡ Dynamic Tables
  
  - Enter number of subjects → rows generated automatically
  - Enter number of semesters → CGPA table generated

- 💡 Clean UI
  
  - Table-based layout
  - Dark theme interface

---

📌 Grading System

Marks (%)| Grade Point
90 - 100| 10
80 - 89| 9
70 - 79| 8
60 - 69| 7
50 - 59| 6
45 - 49| 5
40 - 44| 4
Below 40| 3 (Fail)

---

🧠 How It Works

🔹 SGPA Formula

SGPA = Σ(Credit × Grade Point) / Σ(Credits)

🔹 CGPA Formula

CGPA = Σ(SGPA × Credits) / Σ(Credits)
